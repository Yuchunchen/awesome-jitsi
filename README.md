2020武漢肺炎(COVID-19, 新型冠狀病毒)疫情影響了全世界，「視訊」成為溝通的另外一個重要選項。[Jitsi meet](https://jitsi.org/jitsi-meet/)是一個穩定、安全的開源視訊平台，大家可以依照需求延伸開發更貼身適用客製化視訊系統，在這一波疫情中，被紛紛用來開發疫情視訊系統，這個網頁放了 Jitsi 相關應用，歡迎大家分享更新。
